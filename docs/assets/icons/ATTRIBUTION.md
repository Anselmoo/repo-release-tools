Pinning recommendation
----------------------
For reproducible attribution, pin the Simple Icons commit SHA used when the icons were fetched. Pinning requires the GitHub API or the rrt MCP server (see below). If you want, the maintainer can pin SHAs for the set of icons used and update this file.

How to pin using rrt MCP server
--------------------------------
1. Install the rrt MCP extra and start the server (see .github/skills/rrt-user-mcp-server/SKILL.md).
2. Query the Simple Icons repo commit or file content via the MCP or GitHub API and include the commit SHA in this file.
