# rrt ci-version

Emit the current project version for use in CI pipelines.

## Overview

`rrt ci-version` reads the resolved project version from the configured version
targets and prints it to stdout, making it easy to consume in shell scripts,
GitHub Actions expressions, or other CI steps.
