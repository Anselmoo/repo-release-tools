# rrt env

Display the resolved runtime environment information.

## Overview

`rrt env` prints the Python version, interpreter path, and other environment
details that rrt resolved at runtime. Useful for debugging unexpected behaviour
when multiple Python versions or virtual environments are active.

> **Note:** This page is a stub. Run `rrt env --help` or see the generated
> [CLI reference](rrt-cli.md) for the authoritative command reference.

## Basic usage

```bash
rrt env
```
