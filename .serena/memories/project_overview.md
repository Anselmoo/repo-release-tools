repo-release-tools is a typed Python 3.12+ CLI package for release policy automation in Git repositories. It provides the `rrt` CLI for branch/version/release workflows, `rrt-hooks` for pre-commit/lefthook/CI enforcement, and a reusable GitHub Action. Main modules live in `src/repo_release_tools/`, with hook logic in `hooks.py`, changelog logic in `changelog.py`, config loading in `config.py`, and command implementations under `commands/`. Relevant docs live in `docs/`, including hook setup, semantic branches, GitHub Action usage, and CLI reference.
