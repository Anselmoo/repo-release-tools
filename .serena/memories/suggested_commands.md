Development commands: `uv sync --all-groups` to install dependencies, `uv run pytest -q -m "not runtime"` for fast tests, `uv run pytest -q -m runtime tests/test_runtime_hybrid.py` for runtime integration tests, and `uvx pre-commit run --all-files` for linting/formatting checks. Useful entrypoints: `rrt ...` for the main CLI and `rrt-hooks ...` for hook runners. Common repo navigation commands on Darwin include `git`, `ls`, `cd`, `find`, and `rg`.
