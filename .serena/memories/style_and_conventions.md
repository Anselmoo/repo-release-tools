The project is a typed Python package (`Typing :: Typed`) targeting Python 3.12+ and uses Ruff with line length 100. Release-policy conventions include Conventional Commit types such as feat/fix/docs/chore/ci/build/test/deps, branch names in `<type>/<kebab-slug>` form, and Keep a Changelog with an `[Unreleased]` section. Hook workflows distinguish `incremental` changelog handling from `squash` workflows; `rrt-update-unreleased` auto-manages changelog bullets for incremental repos.
